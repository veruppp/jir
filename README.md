# V-Cloud (Next.js)

Website cloud hosting sederhana untuk upload dan embed video, audio, dan gambar.

## Fitur

- Upload file (video, audio, gambar)
- Link embed pendek: `/e/[slug]`
- Galeri publik
- Tanpa login
- Batas maksimal 10GB
- UI biru-putih elegan

## Deploy ke Vercel

1. Upload ke GitHub
2. Buka [vercel.com/import/git](https://vercel.com/import/git)
3. Import repository ini
